What is Call Back Function? Works of Call Back Function

A callback function is a function that is passed as an argument to another function. Examples : setTimeout(function() { console.log('This code will be executed after 2 seconds'); }, 2000); ** The main purpose of a callback function is to enable asynchronous behavior in a program. This means that instead of waiting for a certain task to complete before moving on to the next task, a program can continue to execute while waiting for a response from an asynchronous task.
